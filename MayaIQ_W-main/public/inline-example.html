How It Works

Simply include the script and create a chat instance in any container element. The chat becomes part of your page flow, not an overlay.

You can place chat widgets in sidebars, content areas, modal dialogs, or anywhere else in your layout. They behave like regular page elements.

ðŸ’» Integration Examples

1. Auto-Initialize with Data Attributes

The simplest way - just add data attributes to any div:

<div data-pingbash-chat 
     data-group-name="support"
     data-api-url="https://api.pingbash.com"
     data-height="500px"
     data-theme="light">
</div>

2. JavaScript API

Create chat instances programmatically:

// Include the script
<script src="/components/WebComponent/PingbashChatInline.js"></script>

// Create chat instance
const chat = PingbashChat.create('#my-chat-container', {
    groupName: 'support',
    apiUrl: 'https://api.pingbash.com',
    theme: 'dark',
    height: '500px',
    customColors: {
        primary: '#ff6b6b',
        secondary: '#4ecdc4'
    }
});

3. Multiple Chat Rooms

You can have multiple chat rooms on the same page:

<div class="support-chat" data-pingbash-chat data-group-name="support"></div>
<div class="sales-chat" data-pingbash-chat data-group-name="sales"></div>
<div class="general-chat" data-pingbash-chat data-group-name="general"></div>
